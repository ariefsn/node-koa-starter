# Node Koa Starter

> Node JS starter with Koa and Typescript

- Clone

  ```shell
  git clone https://github.com/ariefsn/node-koa-starter.git
  ```

- Install

  ```shell
  yarn install
  ```

- Dev

  ```shell
  yarn dev
  ```

- Nodemon

  ```shell
  yarn nodemon
  ```

- Build

  ```shell
  yarn build
  ```

- Start

  ```shell
  yarn start
  ```
